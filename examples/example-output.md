## 9. 结构化 UI 设计说明书

### 9.1 全局规则

- 页面沿用既有后台主布局：Sidebar `220px`，Header `48px`
- 内容区背景为 `#F2F3F5`
- 卡片常规内边距使用 `20px`
- 输入类控件边框采用项目统一覆盖值，不保留 Arco 默认边框色
- Figma 中的 `font-family` 仅作参考，不直接落地到代码

### 9.2 页面整体结构

- 页面主体为单列卡片流布局
- 卡片按“概要 -> 基础信息 -> 经营信息 -> 证照信息 -> 审核备注”自上而下排列
- 底部审核操作栏视觉上吸底，但结构上属于内容区内部子节点
- 内容区为主要滚动容器，页面壳本身不滚动

### 9.3 关键模块详细规格

#### 顶部概要卡

- 高度：`112px`
- 内边距：`20px 24px`
- 布局角色：左固定中间主体右操作
- 标题字号 / 字重 / 行高：`20 / 600 / 28`
- 描述文字字号 / 行高：`14 / 22`

#### 证照信息卡

- 缩略图尺寸：`120 x 90`
- 缩略图间距：`12px`
- 点击缩略图打开图片预览浮层
- hover 时出现遮罩层和“查看大图”提示

#### 底部审核操作栏

- 高度：`72px`
- 内边距：`16px 24px`
- 按钮间距：`12px`
- 跟随内容区宽度，不应直接铺满 viewport
- 需要在内容区主容器上建立定位上下文

## 11. Arco 默认样式覆盖清单

- `Input` / `Textarea`：
  - 边框色覆盖为 `rgba(201, 205, 212, 1)`
  - focus 边框改为 `rgb(0, 170, 166)`
- `Button`：
  - 校正默认高度、左右 padding、圆角与字重
- `Modal`：
  - 校正头部、内容区、底部按钮区间距
- `Tag`：
  - 校正默认 `line-height` 与左右 padding

## 12. 精确定位与 z-index 风险点

- 底部审核操作栏：
  - 高风险点：容易被误实现为 viewport 级 `fixed`
  - 正确做法：优先按内容区内部的定位子节点处理
- 审核驳回弹窗：
  - 需确认遮罩层和层级是否与全局弹窗容器一致
- 图片预览浮层：
  - 需确认和其他全局浮层同时存在时的层级关系

## 13. 视觉归属 / DOM 挂载归属 / 定位参考归属说明

### 底部审核操作栏

- 视觉归属：页面底部操作区

- 定位参考归属：内容区主容器

- 是否影响滚动区：是，需要为内容区正文预留底部安全空间
